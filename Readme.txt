J2プログラミング2 春休み課題
j1714 小山裕

シミュレーションRPG的なゲームを作りました
カーソルやキャラは十字キーで移動
決定は基本的にEnterキーです
タイトル→セレクト画面へ移動しBATTLE1を選ぶとゲームスタートです
※BATTLE2は未作成です

ゲームは大きく分けて移動と攻撃のフェイズに分かれています
移動ではまずサイコロを振り、出た目の分だけ進むことができます
移動を終えた際、自軍のキャラの攻撃範囲(黄色)に敵がいるときや、
逆に敵の攻撃範囲(赤)に入ってしまっているとバトルが始まります。
バトルで敵のHPを減らしていき、敵を全滅させればクリアです
ただし、味方キャラの王女(ピンク髪のキャラ)がやられてしまうと
ゲームオーバーなので気をつけてください
1キャラの行動が終わると1ターン経過します
また、ターンが回ってくる順番は一定となっています
味方キャラが死んだ際、本来そのキャラが行動するはずだったターンが
回ってくると次のキャラのターンへ飛びますが、ターンは+1されます
なるべく少ないターンで敵を全滅させれるよう頑張ってください

ゲームの特徴としてキャラによってステータスやサイコロの出る目が違います
一部のキャラには特殊な能力があったりします
例えば、プータローというキャラは自分が行動する代わりに、任意のキャラの
進める歩数を+６します
またゲームマップの上から3行は"ツタゾーン"といい、そこにいる状態だと
移動の際に進める歩数が4以上でないとそのターンは動けません
そうならないようにプータローの能力を使ったり、味方を押して動かしたりするなど
うまく工夫をしてみてください
